# 资产测绘

```
用于日常工作中的很繁琐的资产测绘任务，被逼疯了，只能自己折腾一个了
```

### 0x01 使用环境
* python3.7+
* 安装好 nmap
* 按照好 masscan

#### 0x02 使用方法
* 运行 main.py 打开对应的8000端口,默认账号密码admin:admin123登录
